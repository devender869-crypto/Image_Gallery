# Stillform Image Gallery

A simple and responsive image gallery website created using HTML, CSS, and JavaScript.

## Features

- Responsive gallery layout for desktop, tablet, and mobile
- Category filters for Nature, Architecture, and People
- Hover effects and smooth image transitions
- Lightbox view for opening images in a larger size
- Next and previous buttons in the lightbox
- Keyboard controls:
  - Left arrow: previous image
  - Right arrow: next image
  - Escape: close lightbox

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Open `index.html` using Live Server, or double-click the file to open it in a browser.

## Note

The gallery images are loaded from Unsplash, so an internet connection is required.
